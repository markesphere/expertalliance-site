# ExpertAlliance.ai Marketing Site

Standalone marketing site for the ExpertAlliance.ai company and the RFPEZ AI app.

## Local preview

### Option A: Python

```bash
cd expertalliance-site
python -m http.server 5175
```

Open `http://localhost:5175`.

### Option B: Node

```bash
cd expertalliance-site
npx --yes serve -l 5175 .
```

## Edit

- Main page: `index.html`
- Styling: `styles.css`

## Repo setup

This folder is intended to become its own git repository.

```bash
cd expertalliance-site
git init
```

Then create a new remote repo (e.g. `expertalliance-site` / `expertalliance.ai`) and push.
